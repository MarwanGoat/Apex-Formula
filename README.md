**Apex Formula** is a high-speed, top-down arcade racing game built entirely in vanilla JavaScript using the HTML5 Canvas API. Race against AI, manage your vehicle integrity, and utilize your KERS battery to secure victory.

## ğŸ® Features

* **Dynamic Physics Engine:** Drift mechanics, collision detection, and surface friction (asphalt vs. grass).
* **Vehicle Systems:**
    * **Health/Integrity:** Taking damage reduces integrity. Hitting 0% destroys the car.
    * **KERS (Kinetic Energy Recovery System):** Strategic nitro boost management.
* **AI Opponents:** Race against 3 AI drivers with adjustable difficulty levels.
* **Customization:** Select from multiple car skins (Standard, Doge, Pepe, Nyan).
* **Leaderboard:** Local storage saves your best lap times.
* **Audio Engine:** Synthesized engine sounds and crash effects using the Web Audio API (no external files required).
